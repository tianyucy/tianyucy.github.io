使用树莓派zero2w驱动屏幕播放视频

2026-04-05
english

介绍

本文章通过python来驱动1.8寸TFT屏幕（使用ST7735S芯片），仅能实现将mp4视频在屏幕上播放。

使用说明

硬件连接

根据您提供的 LCD 引脚定义（GND, VCC, SCL, SDA, RES, DC, CS, BL），请按以下方式连接硬件

GPIO 引脚

LCD 引脚说明 树莓派引脚 (物理编号) 树莓派功能
GND 接地 Pin 6 / 9 / 14 等 GND
VCC 电源 (3.3V) Pin 1 或 Pin 17 3.3V Power
SCL SPI 时钟 (CLK) Pin 23 GPIO 11 (SPI0_SCLK)
SDA SPI 数据 (MOSI) Pin 19 GPIO 10 (SPI0_MOSI)
RES 复位 (Reset) Pin 22 GPIO 25 (可自定义)
DC 数据/命令选择 Pin 18 GPIO 24 (可自定义)
CS 片选 (Chip Select) Pin 24 GPIO 8 (SPI0_CE0)
BL 背光控制 Pin 12 或 Pin 1 GPIO 18 或 3.3V

查看引脚指令

pinout

如何使用

  1. 在终端输入：sudo raspi-config。
  2. 选择 Interface Options -> SPI -> Yes。
  3. 重启设备
  4. 在终端输入 python3 play_video.py 视频名.mp4

import cv2
import time
from luma.lcd.device import st7735
from luma.core.interface.serial import spi
from PIL import Image

# 1. 极限硬件初始化 (60MHz)
serial = spi(port=0, device=0, gpio_DC=24, gpio_RST=25, baudrate=60000000)
# 根据你之前的调试结果，保持 bgr=True 或 False
try:
    device = st7735(serial, width=160, height=128, rotate=0, bgr=False,h_offset=1,v_offset=2)
except Exception as e:
    print(f"FALSE")

def play_video_fast(video_path):
    cap = cv2.VideoCapture(video_path)
    if not cap.isOpened():
        return

    # 获取视频帧率
    fps = cap.get(cv2.CAP_PROP_FPS)
    frame_duration = 1.0 / fps
    
    print(f"原始帧率: {fps}, 目标分辨率: {device.width}x{device.height}")
    
    start_time = time.time()
    frame_count = 0

    try:
        while cap.isOpened():
            ret, frame = cap.read()
            if not ret: break

            frame_count += 1
            
            # --- 核心优化：动态跳帧 ---
            # 计算当前应该播放哪一帧
            elapsed = time.time() - start_time
            expected_frame = int(elapsed / frame_duration)
            
            # 如果当前帧落后于预期进度，跳过这一帧不显示，直接读下一帧
            if frame_count < expected_frame:
                continue

            # --- 极速图像处理 ---
            # 1. 缩小图片尺寸（在转颜色前缩小更省 CPU）
            small_frame = cv2.resize(frame, (device.width, device.height), interpolation=cv2.INTER_NEAREST)
            
            # 2. 颜色转换 (BGR -> RGB)
            rgb_frame = cv2.cvtColor(small_frame, cv2.COLOR_BGR2RGB)
            
            # 3. 直接转 PIL 图像并显示
            img = Image.fromarray(rgb_frame)
            device.display(img)

            # 稍微控制一下，防止跑太快（可选）
            # time.sleep(0.001)

    except KeyboardInterrupt:
        print("停止播放")
    finally:
        cap.release()

if __name__ == "__main__":
    import sys
    video = sys.argv[1] if len(sys.argv) > 1 else "video.mp4"
    play_video_fast(video)
