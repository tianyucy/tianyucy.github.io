Using Raspberry Pi Zero 2W to drive a screen to play videos

2026-04-05
简体中文

Introduction

This article demonstrates how to drive a 1.8-inch TFT screen (using the ST7735S chip) using Python, enabling the playback of mp4 videos on the screen.
The code implementation is achieved through Gemini AI.

Instructions for Use

Hardware connection

According to the LCD pin definitions you provided (GND, VCC, SCL, SDA, RES, DC, CS, BL), please connect the hardware as follows

GPIO pin

LCD Pin Description Raspberry Pi Pins (Physical Number) Raspberry Pi Functions
GND GND Pin 6 / 9 / 14 GND
VCC VCC (3.3V) Pin 1 or Pin 17 3.3V Power
SCL SPI CLK Pin 23 GPIO 11 (SPI0_SCLK)
SDA SPI MOSI Pin 19 GPIO 10 (SPI0_MOSI)
RES Reset Pin 22 GPIO 25(customizable)
DC Data/Command Selection Pin 18 GPIO 24 (customizable)
CS Chip Select Pin 24 GPIO 8 (SPI0_CE0)
BL backlight control Pin 12 or Pin 1 GPIO 18 or 3.3V

How to use

  1. Enter sudo raspi-config in the terminal.
  2. Select Interface Options -> SPI -> Yes.
  3. Restart the device.
  4. Enter python3 play_video.py video_name.mp4 in the terminal.

import cv2
import time
from luma.lcd.device import st7735
from luma.core.interface.serial import spi
from PIL import Image

# 1. 极限硬件初始化 (60MHz)
serial = spi(port=0, device=0, gpio_DC=24, gpio_RST=25, baudrate=60000000)
# 根据你之前的调试结果，保持 bgr=True 或 False
try:
    device = st7735(serial, width=160, height=128, rotate=0, bgr=False,h_offset=1,v_offset=2)
except Exception as e:
    print(f"FALSE")

def play_video_fast(video_path):
    cap = cv2.VideoCapture(video_path)
    if not cap.isOpened():
        return

    # 获取视频帧率
    fps = cap.get(cv2.CAP_PROP_FPS)
    frame_duration = 1.0 / fps
    
    print(f"原始帧率: {fps}, 目标分辨率: {device.width}x{device.height}")
    
    start_time = time.time()
    frame_count = 0

    try:
        while cap.isOpened():
            ret, frame = cap.read()
            if not ret: break

            frame_count += 1
            
            # --- 核心优化：动态跳帧 ---
            # 计算当前应该播放哪一帧
            elapsed = time.time() - start_time
            expected_frame = int(elapsed / frame_duration)
            
            # 如果当前帧落后于预期进度，跳过这一帧不显示，直接读下一帧
            if frame_count < expected_frame:
                continue

            # --- 极速图像处理 ---
            # 1. 缩小图片尺寸（在转颜色前缩小更省 CPU）
            small_frame = cv2.resize(frame, (device.width, device.height), interpolation=cv2.INTER_NEAREST)
            
            # 2. 颜色转换 (BGR -> RGB)
            rgb_frame = cv2.cvtColor(small_frame, cv2.COLOR_BGR2RGB)
            
            # 3. 直接转 PIL 图像并显示
            img = Image.fromarray(rgb_frame)
            device.display(img)

            # 稍微控制一下，防止跑太快（可选）
            # time.sleep(0.001)

    except KeyboardInterrupt:
        print("停止播放")
    finally:
        cap.release()

if __name__ == "__main__":
    import sys
    video = sys.argv[1] if len(sys.argv) > 1 else "video.mp4"
    play_video_fast(video)
